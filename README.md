# Setup and Installation
### 1. Run this command in root of project
```
python3 -m venv venv
source venv/bin/activate
./scripts/setup
```
-  create python virtual env
-  install poetry
-  install packages

### 2. Compile CSS
```
./scripts/npm run tw:watch
```
